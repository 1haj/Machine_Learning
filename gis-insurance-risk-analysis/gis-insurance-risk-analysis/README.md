# GIS-Focused Insurance Risk Analysis

This project demonstrates an end-to-end GIS-focused data science workflow
supporting insurance risk assessment using spatial data, ETL pipelines,
statistical analysis, and machine learning.

## Key Skills Demonstrated
- GIS & spatial analytics (QGIS, GeoPandas)
- PostGIS spatial storage
- Python & Jupyter notebooks
- ETL and data lifecycle management
- Predictive modelling for insurance risk
- Business-oriented insights and decision support
